# Contributing

Thanks for helping improve this project.

## How to contribute
1. For larger changes, open an issue first to align on direction.
2. Branch from `main`, keep commits focused, and open a pull request.
3. Match existing code style and add or update tests when behavior changes.

## Local development
See the repository README for setup, test commands, and project layout.

## Questions and bugs
Use [GitHub Issues](https://github.com/erron-ai/workflow-migrate/issues).
